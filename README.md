
### Installation

1.  Clone the repo:
    
    bash
    
    CopyEdit
    
    `git clone https://github.com/aniket30i/sodio-book-asgn.git cd sodio-book-asgn`

    bash
    
    CopyEdit
    
    `npm install  or yarn install` 
    

### Usage

Run the JSON Server - binded to port 3006

`npm run start:server`



Run the development server:


`npm run dev`
