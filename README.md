### Installation

1.  Clone the repo:
    
    bash
    
   
    
    `git clone https://github.com/aniket30i/sodio-book-asgn.git
    cd sodio-book-asgn` 
    
3.  Install dependencies:
    
    bash
    
   
    
    `npm install  or yarn install` 
    

### Usage

Run the JSON Server - binded to port 3006

`npm run start:server`



Run the development server:


`npm run dev`
